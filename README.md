USpeedo SDK is a Python client library for accessing the USpeedo API.

## Installation

```bash
pip install uspeedo-sdk-py
```

## Migration from uspeedo-sdk-python

This package is the official continuation of `uspeedo-sdk-python`. If you were using the old package, switch to the new one:

```bash
pip uninstall uspeedo-sdk-python
pip install uspeedo-sdk-py
```

The API remains compatible; only the package name has changed.

This client can run on Linux, macOS and Windows.
